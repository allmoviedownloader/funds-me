# Funds Me

Automated Funding Scraper and Dashboard for global startup opportunities.

## Project Structure
- **/frontend**: Premium iOS-style dashboard (HTML/CSS/JS).
- **/backend**: Python scraper using BeautifulSoup and Gemini AI.

## Quick Start
1. Run `backend/scraper.py` to fetch and sync data to Supabase.
2. Open `frontend/index.html` to view the dashboard.

*Automatically deletes expired entries based on deadlines.*
